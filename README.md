# Hướng dẫn cài đặt dự án

## Lưu ý
- Sau khi **pull code** về, bạn cần tạo một file `.env` mới.  
- Copy nội dung từ file `.env.example` vào file `.env`.  
- **Không được ghi đè trực tiếp** vào file `.env.example`.  

---

## Setup Cloudinary

1. Cập nhật file `.env` mới từ file `.env.example`.  
2. Truy cập [Cloudinary](https://cloudinary.com/users/register_free) và đăng ký tài khoản.  
3. Sau khi đăng ký, tại giao diện **API Keys**, bạn sẽ nhận được:  
   - **Cloud name**  
   - **API Key**  
   - **API Secret**  

   ![Cloudinary API Keys](https://github.com/user-attachments/assets/1607dec0-dd5a-40d8-a9a2-f588143d88a3)

4. Thay thế các giá trị này vào file `.env`.
